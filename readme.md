一个专注于2026年美加墨世界杯的微信小程序，提供赛程查询、比分预测、球队分析、冠军预测等功能。

## 项目概述

本小程序为2026年世界杯（6月11日-7月19日）提供全方位的信息服务，包括实时赛程、比分预测、球队数据、积分榜、冠军预测等核心功能。采用微信小程序原生框架开发，支持自定义TabBar、自定义导航栏等特性。

### 核心特性

- **实时赛程**：按日期查看比赛安排，支持日历筛选
- **比分预测**：用户可预测比赛比分，查看预测分布
- **冠军预测**：支持"命运抽奖"和"亲选冠军"两种方式
- **球队数据**：32支参赛球队详细信息、历史战绩
- **积分榜**：12个小组的实时积分榜
- **对阵图**：完整的小组赛和淘汰赛对阵图
- **个人中心**：收藏比赛、预测记录、分享海报

## 自定义组件

### 1. `custom-nav`
自定义导航栏组件，支持返回按钮、标题、收藏按钮、透明背景等。

### 2. `custom-tab-bar`
自定义TabBar组件，支持4个标签页：首页、赛程、积分榜、我的。

### 3. `match-card`
比赛卡片组件，显示球队信息、比分、状态、预测数等。

### 4. `team-flag`
球队旗帜组件，支持emoji和图片两种显示方式。

### 5. `countdown`
赛事倒计时组件，显示距离开赛/结束的时间。

### 6. `pill-tab`
胶囊标签组件，用于日期选择、筛选标签等。

### 7. `score-input`
比分输入组件，支持加减按钮输入比分。

### 8. `globe`
3D地球组件，用于冠军球队选择。
